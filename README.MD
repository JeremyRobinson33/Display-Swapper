# Window Swapper

## A tool to swap all windows from one monitor to another in a 2 Monitor Setup

### 1. Download/clone the project

### 2. Edit run_cmd.bat

#### &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 2a. set the first command to be the path to your python executable

#### &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 2b. set the second command to be the path to the windowSwapper.py

### 3. Using a keymapper, such as logitech g hub, map a key to the run_cmd.bat to swap displays on the fly
